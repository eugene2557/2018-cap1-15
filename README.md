

### 1. 프로잭트 소개


정적인 전시물을 VR 과 AR을 통하여 생동감있게 체험하고 학습한다.

소비자는 흥미유발, 전시물에 대한 지식 습득, 거리의 제약 극복이란 이점이 있고

박물관 측은 소비자의 관심확대로 인한 홍보 효과를 누릴 수 있다



개발도구 : 유니티
